1. What is DOM and what is the purpose of DOM?

DOM represents for Document Object Model.
It is a programming interface for web documents.
It gives a logical structure of documents and the way it is manipulated.
It acts as a tree of objects. We can change HTML elements, attributes,
add new and remove elements, style the elements. We can also add events to the
page using DOM. We use the document word to define the document object.

2. How will you select HTML elements using DOM? Name the DOM methods.

We can select elements by using selectors.
DOM selectors selects HTML elements within document.
After writing document, we can select from any of the six methods:

  • getElementByTagName()
  • getElementById()
  • getElementByClassName()
  • getElementByName()
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Event bubble is a concept of DOM. It happens when an element receives
an event and that event bubbles up to its parents and anscestors until it
reaches the root element. To stop event bubbling, we use event.stopPropagation().

4. What is a callback function and why will you use it?

A Callback function is a function that is passed as an argument.
We need callback functions because JS runs code in a top-down fashion.
JS actions don't stop the program from running until they are completed.
Callback makes sure that a function is not going to run before a task is
completed and only run when the current program is finished. Hence the
sequence of code is maintained and errors are avoided.
